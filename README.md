# 🚀 Drokia Trends Pro (V2.0)

Advanced Telegram Voting and Trending Bot for Crypto Communities.

**Drokia Trends Pro** is a high-performance, automated Telegram voting and trending bot designed for Web3 and crypto communities. It provides real-time tracking, sybil-resistant voting mechanics, and seamless integration with live market data.

## 🌟 Core Features
* **DexScreener API Integration:** Automatically fetches and verifies token data (Name, Symbol) directly from smart contracts (CA).
* **Live Leaderboard:** Maintains a real-time "TOP 10 Drokia Trending Gems" list, updating dynamically based on community votes over a 12-hour rolling window.
* **Sybil-Resistant Voting System:**
  * Strict 1-hour cooldown per user per project.
  * Mandatory Official Channel subscription requirement to cast a vote.
